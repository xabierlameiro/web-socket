# WebSocket Server

A Node.js WebSocket server demonstrating real-time bidirectional communication,
deployable to Heroku.

[![CI](https://github.com/xabierlameiro/web-socket/actions/workflows/ci.yml/badge.svg)](https://github.com/xabierlameiro/web-socket/actions/workflows/ci.yml)

## Stack

| Layer      | Choice    |
| ---------- | --------- |
| Runtime    | Node.js   |
| Protocol   | WebSocket |
| Deployment | Heroku    |

## Getting Started

```bash
npm install
npm start
```

## Deploy to Heroku

```bash
heroku create
git push heroku master
heroku open
```
